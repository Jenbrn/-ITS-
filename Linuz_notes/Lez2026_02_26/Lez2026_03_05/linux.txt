simmetrica : si cifra e decifra con la stessa chiave. Meno sicuro di c asimmetrica.
asimmetrica: disaccoppiamento cifratura e decifratura. Cifro con chiave pubblica dell'altra parte, per decriptarle sarà necessaria la chiave privata del 'ricevente'.

installazione terminale
-sudo apt update : aggiorno repo
-sudo apt install openssh-server : installo un programma, in questo caso openssh-server
-ps aux : vedo i processi al momento attivi
-ps aux | grep ssh : cerco un processo nel dettaglio usando grep
- sudo systemctl satus ssh : chiedo lo status di un servizio
- sudo systemctl start ssh : avvio il programma
- sudo systemctl sto ssh : arresto
- sudo systemctl disable ssh : disabilitato
sudo systemctl enable ssh
* preset -> di default 
* enable se si ha  necessità che si avvii da solo
*daemon -> = servizio -> d
