---------------networking-----
- eth0 - enp3s0 -> ethernet fisica
- wlan0 - wlp2s0 -> wireless
- lo -> loopback (servizi che comunicano localmente)
-vibr0 - docker0 -> bridge virtuale

- ip (ifconfig) :
* visualizzazione config rete
- ip addr show -> indirizzi assegnati interfacce
- ip addr show enp3s0 -> interfaccia specifiva
- ip route show -> rotte configurate
- ip link show -> interfacce e il loro stato (up-Down)
- ip addr add | addr del | link set -> mod temporanee

* modifiche temporanee - agg o rem ip o mod stato interfacce
- ip addr add ip dev enp3s0 -> add ip
- ip addr del ip dev enp3s0 -> rm ip
- ip link set ip enp3s0 up / down -> attiva o disattiva interfaccia

* routing gestione rotte
- ip route add default via ip -> imposto gw predefinitp
- ip route add ip via ip -> aggiunge rotta verso rete specifica
- ip route del default via ip -> cancella rotta di def

-------------------persistenza------------
- /etc/network/interfaces -> file testuale dove definisco interfacce e parametri per persistenza es:
    auto enp3s0
    iface enp3s0 inet static
    address 192.168.1.100
    netmask 255.255.255.0
    gateway 192.168.1.1
    dns-nameservers 8.8.8.8 1.1.1.1
    seg da systemctl restart networking
- /etc/hosts -> associare nomi a ip localmente 
- /etc/resolv.com -> indirizzi server dns per risoluz
- tcpdump -i enp3s0 -> per analisi problemi di rete, motra in tempo reale i pacchetti che passano sull'interfaccia
* può essere combinato ad una specifica porta
- traceroute -> traccia percorso pacchetti
