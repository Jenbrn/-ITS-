- R -> running
- S -> sleeping in attesa evemto
- D -> Disk sleep attesa i/o
- Z -> zombie terminato ma non rimosso
- T -> stopped 
- I -> Idle

- ps aux -> processi attivi 
* pr basso priorità alta
* ni -> nice value -20 max +19 min

- killall nginx -> tutti i processi con un nome
