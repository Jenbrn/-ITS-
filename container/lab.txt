1. Configurare AWS CLI  →  autenticarti da terminale
2. Creare repository ECR  →  il "magazzino" dove vive la tua immagine
3. Autenticare Docker su ECR  →  dare a Docker il permesso di caricare
4. Build e Taggare l'immagine  →  rinominarla nel formato che ECR si aspetta
5. Push dell'immagine su ECR  →  caricamento vero e proprio
6. Creare il cluster ECS  →  l'ambiente dove girerà il container
7. Creo Securety group -> definiamo connessioni possibili in entrata e uscita
8. Creare Task Definition  →  dire ad ECS come avviare il container
9. Avviare il servizio
